## Bài 5: In ra N dấu *

Nhập vào một số N (1 <= N <= 1000). In ra N dấu *

**Đầu vào:**

- Số nguyên N (1 <= N <= 1000)

**Đầu ra:**

In ra N dấu *

### Ví dụ

#### Ví dụ 1

| Input | Output |
|-------|--------|
| 3     | *** |

#### Ví dụ 2

| Input | Output |
|-------|--------|
| 5     | ***** |
