### Bài 4: Tính toán 2 số

Hãy nhập vào 2 số nguyên và in ra màn hình tổng, hiệu, tích, thương của 2 số đó.

**Đầu vào:**

- Hai số nguyên `a`, `b`

**Đầu ra:**

In ra màn hình các phép toán cơ bản giữa `a` và `b` (xem ở ví dụ)

#### Ví dụ:

##### Ví dụ 1:

| Input | Output                  |
|-------|-------------------------|
| 5     | a + b = 5               |
| 3     | a - b = 2               |
|       | a * b = 15              |
|       | a / b = 1 |

##### Ví dụ 2:

| Input | Output                  |
|-------|-------------------------|
| 10    | a + b = 10              |
| 2     | a - b = 8               |
|       | a * b = 20              |
|       | a / b = 5               |
