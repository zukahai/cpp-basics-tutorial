# Ký tự và chuỗi

Bảng mã **ASCII**

<p align="center"> <img src="./images/ascii.png" alt="bg" /> </p>

- Cách chuyển ký tự thành số trong C++

```cpp
char c = 'A';
int n = c;
cout << n; // 65
```

- Cách chuyển số thành ký tự trong C++

```cpp
int n = 65;
char c = n;
cout << c; // A
```
