
Python

```python

a = 10

if a > 5:
    print("a > 5")
    print("a = ", a)


```




C++

```cpp

int a = 10;

if (a > 5) {
    cout << "a > 5" << endl;
    cout << "a = " << a << endl;
}

```

- Điều kiện phải đặt trong dấu **(điều kiện)**
- Khối lệnh phải đặt trong dấu **{khối lệnh}**

- **and** chuyển thành **&&**
- **or** chuyển thành **||**
- **elif** chuyển thành **else if**
