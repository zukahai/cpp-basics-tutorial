Nhập vào a. Tính giá trị

1 * a + 2 * (a - 1) + 3 * (a - 2) + ... + a * 1

Ví dụ:

a = 5

1 * 5 + 2 * 4 + 3 * 3 + 4 * 2 + 5 * 1 = 35
